## Usage

```bash
mkdir -p ~/.local/share/gnome-shell/extensions/lovetty@azzamsa.com
cd ~/.local/share/gnome-shell/extensions/lovetty@azzamsa.com
git clone https://github.com/azzamsa/lovetty --depth 1
```

## Origin of the name

The name Lovetty is a played version of Love and Confetti.
